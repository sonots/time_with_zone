TimeWithZone

Handle time with zone without ActiveSupport or ENV['TZ'] for thread-safety.

Installation

Add this line to your application's Gemfile:

gem 'time_with_zone'

And then execute:

$ bundle

Or install it yourself as:

$ gem install time_with_zone
